# rule-of-thump

## Project setup
```
git clone https://github.com/alejandrocepeda/rule-of-thump-api.git
cd rule-of-thump-api
npm install
```

```
cp .env.example .env
```

### Change you .env
```
NODE_ENV=development
DB_HOST=72.55.168.5 
DB_NAME=rule-of-thump
DB_PORT=5433
DB_USER=user
DB_PWD=user
DB_DIALECT=postgres
DB_LOGGING=development
```

### Run project
```
npm run dev
```
